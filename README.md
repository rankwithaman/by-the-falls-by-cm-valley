## 🌊 Overview

**By The Falls By CM Valley** is a fully responsive, premium restaurant website designed to reflect the luxury, warmth, and natural beauty of a fine-dining experience set beside cascading waterfalls in Ludhiana, Punjab.

The website features cinematic visuals, elegant typography, glassmorphism effects, smooth scroll animations, and a complete multi-page structure — all built with vanilla HTML5, CSS3, and JavaScript. No React, no Node.js, no installation required.

## 👁️ Live Preview

To preview the website locally:

2. Open `index.html` in any modern browser
3. All pages, animations, and interactions work instantly
